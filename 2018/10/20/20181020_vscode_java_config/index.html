VSCode的Java环境配置

发表于 2018-10-20 | 分类于 Coding

前言

最近电脑运行的东西有点多，IDEA又有点重，导致跑一些Java的代码很慢，因此考虑用VSCode搭建一个便捷的Java开发与测试环境。

操作

  • Java环境配置（Skip）
  • VSCode开发插件安装
  • Maven也要用上
  • Git可视化版本管理

VSCode插件安装

  • Java Extension Pack（必备，java扩展包，包含了下面五个扩展）
  • Language Support for Java (必备，支持自动补全、定位等）
  • Debugger for Java（调试与运行）
  • Java Dependency Viewer（依赖查看）
  • Maven for Java
  • Java Test Runner

Maven在轻量开发中的应用

通过”All Commands”操作（默认快捷键是shift+cmd+p）

调用通过Maven Generate from Maven Archetype，可以直接创建项目

  • groupId: 代表组织和整个项目的唯一标志
  • artifactId: 具体项目的名称
  • version: 说明当前的版本
  • packaging: 项目的输出格式

VSCode中的可视化Git操作

  • 文件的历史版本回溯

“shift+ctrl+G” 文件变更情况

  • GitLens插件

查看git管理下仓库、文件的详细信息。
